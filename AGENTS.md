# AGENTS.md

This repository contains a Google Apps Script project.

Rules for Codex:

1. The project is written in Google Apps Script (.gs files).
2. Use DriveApp and SpreadsheetApp APIs.
3. Do not use Node.js libraries.
4. Keep functions small and readable.
5. Prefer logging with Logger.log().
6. Avoid breaking existing functionality.

Typical tasks:

- list files in Google Drive folders
- write results to Google Sheets
- automate Drive and Sheets workflows
